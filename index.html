Secure-Eye

Secure-Eye is my MCA final year project work. Developed using C#. Summary of this project description is to secure the place by placing the ordinary webcam which will take the photo in certin time interval and compares with the previous photo. Then it will run through the image processing algorithm which will output the difference in percentage. If difference is less then 95%, owner of the place is alerted with siren sound and SMS notification.
